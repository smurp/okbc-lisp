Documentation for the OKBC-TEST package

This document contains documentation for the OKBC-TEST package in OKBC, which defines the OKBC test suite for the Lisp implementation. The test suite is expected to be useful primarily to OKBC back end authors.

test (okbc-test::kb-class &key (okbc-test::initargs nil) okbc-test::silent (okbc-test::copy t) (okbc-test::tell-and-ask t) (okbc-test::creator nil))
[function]
Runs a suite of tests on a new kb of type kb-class using :initargs. If :copy is true, then attempts to copy from the kb into a ok-back:tuple-kb (called test-target-kb). If tell-and-ask is true, then the tests will then create another KB of the kb-class and test the tell&ask interface as well. The creator may be a function of one argument, a kb-name, that should return an new kb of the type that is to be tested. This is useful for testing non-standard kbs.
test-kb (kb okbc-test::silent &optional (okbc-test::new-kb nil) (okbc-test::ask-kb nil))
[function]
Runs the test suite on a particular KB. If silent is true, it supresses output. If new-kb is provided, the KB copying tests will be run. If ask-kb is provided, the tell&ask tests will be run.
